Comment naviguer sur notre site

Dans le coin en haut à droite de votre écran vous avez le menu, cliquez dessus et vous allez avoir toutes les pages qui vont s'afficher. Il vous suffira plus qu'à sélectionner la page qu'il vous faut.

Créer votre compte

Si vous voulez vous inscrire, vous pouvez vous créer un compte gratuitement. Il faudra juste attendre qu'un administrateur accepte la demande. Ensuite, vous pourrez personnaliser votre profil avec une biographie, une image de profil, ainsi qu'une bannière. La bannière devra elle aussi être validée par un administrateur.
